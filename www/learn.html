Substance SwingX documentation

To use the Substance SwingX plugin add the matching substance-swingx.jar to the classpath of your application. Substance will pick up this jar at runtime and install the matching UI delegates on the SwingX components used in the application.

Due to the development changes in the SwingX library, use the swingx.jar shipped in the matching distribution archive of Substance SwingX plugin. Otherwise your application may see exceptions, visual artifacts and other problems.
